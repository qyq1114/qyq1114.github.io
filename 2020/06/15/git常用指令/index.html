git常用指令

Posted on 2020-06-15

创建新的Git仓库

$ git init

拷贝一个Git仓库到本地

git clone [url]

添加到缓存

git add

$ git add README hello.php

查看在你上次提交之后是否有修改

将缓存区内容添加到仓库中

git commit

git commit -a 可以跳过git add

推送到远程服务器

git push

取消已缓存的内容

git reset HEAD

创建分支命令

git branch (branchname)

切换分支命令

git checkout (branchname)

合并分支命令

git merge

删除分支命令

git branch -d (branchname)
